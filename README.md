# Sistema de Gerenciamento de Clínica Veterinária

Atividade prática de Arquitetura Hexagonal com Java.

## Discente

Pedro Henrique de Araujo Pereira

## Objetivo

Implementar um sistema Java puro para gerenciamento de consultas veterinárias, seguindo a Arquitetura Hexagonal (Ports and Adapters).

## Estrutura inicial

O projeto será organizado em três áreas principais:

- `dominio`: núcleo da aplicação, sem dependências de infraestrutura ou apresentação.
- `infraestrutura`: adaptadores concretos de persistência e notificação.
- `apresentacao`: composição da aplicação e fluxo de demonstração.

## Compilação e execução

A execução será feita com Java puro. As instruções completas serão atualizadas conforme a implementação avançar.
